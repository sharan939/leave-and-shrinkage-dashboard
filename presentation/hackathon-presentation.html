The Problem

"How many hours does your team spend every week just figuring out capacity?"

3+ hours/week per manager on reporting

5+ disconnected trackers for one team

Zero forecasting — react after the gap hits

No alerts — missed updates go unnoticed

The Solution — All-in-One Dashboard

👥 Org Auto-Sync

Pulls full org tree from Phonetool every 30 days. People join/leave — updates itself.

📅 Leave Calendar

2026 India holidays, weekends (red), holidays (green), individual leaves — one view.

✎ Daily Work Tracker

Associates log program, target, actual, shrinkage in 30 seconds. Auto-alerts for missing entries.

📈 Real-time Shrinkage

Expected vs actual, broken by type: planned, unplanned, mandatory off. Always current.

🔮 3-Month Forecast

Predicts future shrinkage from historical patterns + upcoming festivals. See the future.

📨 One-Click WBR Email

Auto-generates Outlook-ready narrative: summary, program breakdown, individual performance.

📩 Monthly Reports

Auto-sends shrinkage emails: leave patterns, team totals, expected vs actual %.

👥 Multi-Manager View

Any manager sees their team only. Scoped per manager — your team, your data.

How We Built It

Lightweight. Zero infrastructure. Zero cost.

HTML5 + CSS3 Vanilla JavaScript (3700+ lines) LocalStorage (no database) Phonetool JSON API Mailto API (Outlook) GitHub Pages

No servers. No AWS. No pipelines. No dependencies.
Just open the HTML file — works offline, works anywhere.
