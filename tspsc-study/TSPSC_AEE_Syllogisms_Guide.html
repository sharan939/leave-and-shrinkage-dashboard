Part 1: Theory & Rules (Must Learn First)

What is a Syllogism?

A syllogism is a logical argument where a conclusion is drawn from two given statements (premises). You must assume the statements to be TRUE even if they seem absurd in real life.

Example:
Statement 1: All cats are dogs.
Statement 2: All dogs are birds.
Conclusion: All cats are birds. ✅ (This FOLLOWS even though it's absurd in reality)

⚠️ Golden Rule

NEVER use common sense or real-world knowledge. Only use the given statements to draw conclusions.

If the statement says "All men are women" — accept it as TRUE and work with it!

E (Universal Negative)No S is PNo cat is a dogES and P have ZERO overlap
I (Particular Affirmative)Some S are PSome boys are tallIAt least one S is also P
O (Particular Negative)Some S are not PSome birds are not eaglesOAt least one S is outside P
💡 Memory Trick: Remember A E I O from the words AffIrmo (I affirm) and nEgO (I deny).
A & I = Affirmative (positive statements)
E & O = Negative (negative statements)
A & E = Universal (talks about ALL/NONE)
I & O = Particular (talks about SOME)

Part 4: Rules for Drawing Conclusions

🔑 Master Rules (MEMORIZE THESE)

  • Rule 1: All + All = All (A + A = A)
  • Rule 2: All + No = No (A + E = E)
  • Rule 3: All + Some = No conclusion about "All" (A + I = No definite conclusion)
  • Rule 4: Some + All = Some (I + A = I)
  • Rule 5: Some + No = Some...not (I + E = O)
  • Rule 6: Some + Some = No conclusion (I + I = No conclusion)
  • Rule 7: No + No = No conclusion (E + E = No conclusion)
  • Rule 8: No + All = Some...not (REVERSE) (E + A = O reverse)

�� Conversion Rules (What you can derive from a single statement)

Given StatementValid ConversionInvalid Conversion

No S is P (E)No P is S (E) ✅—
Some S are P (I)Some P are S (I) ✅All S are P ❌

⚠️ CRITICAL RULE: "Some S are not P" CANNOT be converted to "Some P are not S". This is the #1 mistake students make!

Example: "Some dogs are not black" does NOT mean "Some black things are not dogs"

🔑 Complementary Pairs (Either-Or Rule)

If neither of two conclusions follows individually, check if they form a complementary pair:

  • I and O are complementary: "Some S are P" AND "Some S are not P"
  • A and O are complementary: "All S are P" AND "Some S are not P"
  • E and I are complementary: "No S is P" AND "Some S are P"

If two conclusions are complementary → "Either I or II follows"

Part 3: Venn Diagram Method (Easiest Method)

Step-by-Step Method to Solve ANY Syllogism:

  1. Read both statements carefully
  2. Draw Venn Diagrams for the statements (draw ALL possible diagrams)
  3. Check each conclusion against ALL possible diagrams
  4. A conclusion follows ONLY if it is true in ALL possible diagrams
  5. If a conclusion fails in even ONE possible diagram, it does NOT follow
Golden Rule: A conclusion is VALID only if there is NO possible Venn diagram where it fails. If you can draw even one diagram where the conclusion is false, then it DOES NOT FOLLOW.

Part 2: Types of Statements - Detailed

Understanding "All S are P" (Type A)

When we say "All S are P", it means:

  • Every single S is definitely a P
  • But NOT all P need to be S (some P may not be S)
  • Conversion: "All S are P" → "Some P are S" ✅ (valid)
  • Wrong conversion: "All S are P" → "All P are S" ❌ (INVALID)
┌─────────────────┐ │ P │ │ ┌───────┐ │ │ │ S │ │ │ └───────┘ │ └─────────────────┘ S is completely inside P

Understanding "No S is P" (Type E)

When we say "No S is P", it means:

  • There is ZERO overlap between S and P
  • No member of S belongs to P, and no member of P belongs to S
  • Conversion: "No S is P" → "No P is S" ✅ (valid - reversible!)
┌───────┐ ┌───────┐ │ S │ │ P │ └───────┘ └───────┘ S and P are completely separate

Understanding "Some S are P" (Type I)

When we say "Some S are P", it means:

  • At least one S is a P (could be 1, could be all - minimum 1)
  • Conversion: "Some S are P" → "Some P are S" ✅ (valid - reversible!)
  • "Some" means AT LEAST ONE
┌───────────┐ │ S ┌───┼────────┐ │ │ X │ P │ │ └───┼────────┘ └───────────┘ X = the "some" that are both S and P

Understanding "Some S are not P" (Type O)

When we say "Some S are not P", it means:

  • At least one S exists that is NOT a P
  • Conversion: "Some S are not P" → CANNOT be converted ❌
  • This is the ONLY type that cannot be simply converted
┌───────────┐ │ X S ┌───┼────────┐ │ │ │ P │ │ └───┼────────┘ └───────────┘ X = the "some" that are S but NOT P

Part 5: Solved Examples - Basic (Q.1-30)

Q.1. Statements: All roses are flowers. All flowers are beautiful.
Conclusions: I. All roses are beautiful. II. Some beautiful things are roses. Ans: BOTH I and II follow
Explanation:
• "All roses are flowers" + "All flowers are beautiful" → All + All = All → All roses are beautiful ✅ (Conclusion I follows)
• Since "All roses are beautiful" is true, we can convert: Some beautiful things are roses ✅ (Conclusion II follows)
• Venn: Roses ⊂ Flowers ⊂ Beautiful (nested circles)
Q.2. Statements: All dogs are animals. All animals are living beings.
Conclusions: I. All dogs are living beings. II. All living beings are dogs. Ans: Only Conclusion I follows
Explanation:
• All dogs are animals + All animals are living beings → All + All = All → All dogs are living beings ✅
• "All living beings are dogs" ❌ - WRONG! Living beings is the BIGGER set. Not all living beings are dogs (cats, birds etc. are also living beings)
• Remember: "All A are B" does NOT mean "All B are A"
Q.3. Statements: No cat is a dog. All dogs are faithful.
Conclusions: I. No cat is faithful. II. Some faithful are not cats. Ans: Only Conclusion II follows
Explanation:
• "No cat is dog" + "All dogs are faithful" → No + All = Some...not (reverse)
• Can we say "No cat is faithful"? ❌ NO! Cats might be faithful through some other route. We only know dogs are faithful, but other things could also be faithful.
• "Some faithful are not cats" ✅ YES! Since all dogs are faithful and no cat is a dog, those faithful things that are dogs are definitely not cats.
• Venn: Cat circle separate from Dog circle; Dog circle inside Faithful circle. Part of Faithful (the dog part) is definitely not Cat.
Q.4. Statements: Some boys are tall. All tall are strong.
Conclusions: I. Some boys are strong. II. All strong are boys. Ans: Only Conclusion I follows
Explanation:
• "Some boys are tall" + "All tall are strong" → Some + All = Some → Some boys are strong ✅
• "All strong are boys" ❌ - We can't say this. Strong is the bigger set; there could be strong people who are not boys.
• Rule applied: I + A = I
Q.5. Statements: Some pens are pencils. Some pencils are erasers.
Conclusions: I. Some pens are erasers. II. Some erasers are pens. Ans: NEITHER conclusion follows
Explanation:
• "Some pens are pencils" + "Some pencils are erasers" → Some + Some = NO CONCLUSION
• KEY RULE: Two "Some" statements NEVER give a definite conclusion!
• The pencils that are pens might be DIFFERENT from the pencils that are erasers. There's no guaranteed overlap.
• Venn: Three circles where Pen-Pencil overlap and Pencil-Eraser overlap, but Pen-Eraser may or may not overlap.
Q.6. Statements: No man is a woman. No woman is a child.
Conclusions: I. No man is a child. II. No child is a man. Ans: NEITHER conclusion follows
Explanation:
• "No man is woman" + "No woman is child" → No + No = NO CONCLUSION
• KEY RULE: Two Negative statements NEVER give a definite conclusion!
• Men could still be children (young boys). The fact that men aren't women and women aren't children doesn't tell us anything about men and children.
• Venn: Man and Woman are separate; Woman and Child are separate; but Man and Child COULD overlap or not.
Q.7. Statements: All teachers are educated. Some educated are rich.
Conclusions: I. Some teachers are rich. II. Some rich are educated. Ans: Only Conclusion II follows
Explanation:
• "All teachers are educated" + "Some educated are rich" → All + Some = No conclusion about teachers-rich
• "Some teachers are rich" ❌ - The "some educated who are rich" might not include any teachers!
• "Some rich are educated" ✅ - This is just the conversion of "Some educated are rich" (Some I → Some I, reversible)
• Remember: "Some A are B" always means "Some B are A" too!
Q.8. Statements: All books are pages. No page is a pen.
Conclusions: I. No book is a pen. II. Some pages are books. Ans: BOTH I and II follow
Explanation:
• "All books are pages" + "No page is pen" → All + No = No → No book is a pen ✅
• "Some pages are books" ✅ - This is conversion of "All books are pages" (All A are B → Some B are A)
• Rule: A + E = E
Q.9. Statements: Some cats are dogs. All dogs are animals.

Part 1: Theory & Rules (Must Learn First)

What is a Syllogism?

A syllogism is a logical argument where a conclusion is drawn from two given statements (premises). You must assume the statements are TRUE even if they contradict real life.

Example:
Statement 1: All cats are dogs.
Statement 2: All dogs are birds.
Conclusion: All cats are birds. ✅ (Valid)

The 4 Types of Statements (MUST MEMORIZE)

TypeFormatExampleSymbol
Universal AffirmativeAll A are BAll dogs are animalsA-type
Universal NegativeNo A is BNo cat is a dogE-type
Particular AffirmativeSome A are BSome boys are tallI-type
Particular NegativeSome A are not BSome girls are not dancersO-type

Golden Rules of Syllogism

  • Rule 1: "All A are B" does NOT mean "All B are A" (only "Some B are A" follows)
  • Rule 2: "Some A are B" also means "Some B are A" (conversion is valid)
  • Rule 3: "No A is B" also means "No B is A" (conversion is valid)
  • Rule 4: "Some A are not B" does NOT mean "Some B are not A"
  • Rule 5: From two Particular statements, NO conclusion follows
  • Rule 6: From two Negative statements, NO conclusion follows
  • Rule 7: If one premise is Particular, conclusion must be Particular
  • Rule 8: If one premise is Negative, conclusion must be Negative

2. "No A is B" → Draw circles A and B completely SEPARATE (no overlap)

┌─────┐ ┌─────┐ │ A │ │ B │ └─────┘ └─────┘ No overlap at all. Nothing common.

3. "Some A are B" → Draw circles A and B OVERLAPPING (partial)

┌─────┐ │ A ├──┐ └──┬──┘ │ B │ ┌──┘ └──┘ Partial overlap. Some elements are common.

4. "Some A are not B" → Part of A is OUTSIDE B

┌─────┐ │ A ├──┐ └──┬──┘ │ B │ ┌──┘ └──┘ Part of A does not overlap with B. (At least one A exists that is not B)
KEY TIP: For "All A are B", draw ALL possible diagrams. Sometimes A=B (both circles same size, overlapping completely). The conclusion must be true in ALL possible diagrams to be valid.

Part 1: Theory & Rules (MUST LEARN FIRST)

What is a Syllogism?

A syllogism is a logical argument where a conclusion is drawn from two given statements (premises). You must determine if the conclusion logically follows from the statements - NOT based on real-world truth.

Key Rule: Forget real-world knowledge. Only use what is given in the statements. If "All cats are dogs" is given, accept it as true!

Part 2: Four Types of Statements

The 4 Standard Forms (Remember: A, E, I, O)

TypeFormExampleVenn Diagram
A (Universal Affirmative)All A are BAll dogs are animalsCircle A completely inside Circle B
E (Universal Negative)No A is BNo cat is a dogCircle A and Circle B completely separate
I (Particular Affirmative)Some A are BSome students are girlsCircle A and Circle B partially overlap
O (Particular Negative)Some A are not BSome birds are not parrotsPart of Circle A is outside Circle B

GOLDEN RULES to Remember:

  • "All A are B" does NOT mean "All B are A" (A is inside B, but B may be bigger)
  • "All A are B" automatically means "Some A are B" and "Some B are A"
  • "Some A are B" automatically means "Some B are A" (conversion is valid)
  • "No A is B" automatically means "No B is A" (conversion is valid)
  • "Some A are not B" CANNOT be converted to "Some B are not A"
  • Two particular (Some) statements give NO definite conclusion
  • Two negative (No/Not) statements give NO definite conclusion

Part 3: Venn Diagram Method (EASIEST METHOD)

How to Solve Using Venn Diagrams:

  1. Step 1: Draw circles for each term based on the statements
  2. Step 2: "All A are B" = Draw A completely inside B
  3. Step 3: "No A is B" = Draw A and B completely separate
  4. Step 4: "Some A are B" = Draw A and B overlapping
  5. Step 5: Check if the conclusion is TRUE in ALL possible diagrams
  6. Step 6: If conclusion fails in even ONE possible diagram, it DOES NOT FOLLOW
PRO TIP: For "Some A are B", draw MULTIPLE possible diagrams (partial overlap, complete overlap, one inside other) and check if conclusion holds in ALL cases.

Part 5: Solved Examples - Basic (Q.1-30)

Q.1. Statements: All dogs are animals. All animals are living beings.
Conclusions: I. All dogs are living beings. II. Some living beings are dogs. Ans: Both I and II follow
Method: Draw Dog circle inside Animal circle, Animal circle inside Living Being circle.
Logic: All A are B + All B are C = All A are C (Type A+A = A)
Dog ⊂ Animal ⊂ Living Being → So Dog ⊂ Living Being (Conclusion I: TRUE)
Since All dogs are living beings → Some living beings are dogs (Conclusion II: TRUE - reverse of universal gives particular)
Answer: Both follow ✓
Q.2. Statements: All cats are dogs. All dogs are birds.
Conclusions: I. All cats are birds. II. All birds are cats. Ans: Only Conclusion I follows
Method: Cat ⊂ Dog ⊂ Bird
I. All cats are birds: Cat is inside Dog, Dog is inside Bird → Cat is inside Bird → TRUE ✓
II. All birds are cats: Bird is the BIGGEST circle. Cat is smallest. Bird is NOT inside Cat → FALSE ✗
Remember: "All A are B" does NOT mean "All B are A"!
Answer: Only I follows ✓
Q.3. Statements: No man is a woman. All women are beautiful.
Conclusions: I. No man is beautiful. II. Some beautiful are not men. Ans: Only Conclusion II follows
Method: Man and Woman are separate circles. Woman is inside Beautiful.
I. No man is beautiful: Man is separate from Woman, but Beautiful is BIGGER than Woman. Man COULD overlap with Beautiful (the part outside Woman). So this is NOT necessarily true → FALSE ✗
II. Some beautiful are not men: Women are beautiful and Women are NOT men. So at least those beautiful things (women) are not men → TRUE ✓
Rule Applied: No A is B + All B are C = Some C are not A
Answer: Only II follows ✓
Q.4. Statements: Some books are pens. All pens are pencils.
Conclusions: I. Some books are pencils. II. All pencils are books. Ans: Only Conclusion I follows
Method: Book and Pen overlap. Pen is inside Pencil.
I. Some books are pencils: The books that are pens → those pens are inside pencils → so those books are also pencils → TRUE ✓
II. All pencils are books: Pencil is bigger than Pen. Books only overlap with Pen (partially). Pencil has parts that are NOT books → FALSE ✗
Rule Applied: Some A are B + All B are C = Some A are C
Answer: Only I follows ✓
Q.5. Statements: All roses are flowers. Some flowers are red.
Conclusions: I. Some roses are red. II. Some red are flowers. Ans: Only Conclusion II follows
Method: Rose is inside Flower. Red overlaps with Flower.
I. Some roses are red: Red overlaps with Flower, but the overlap might be in the part of Flower that is OUTSIDE Rose. We can't be sure → FALSE ✗
II. Some red are flowers: "Some flowers are red" can be directly converted to "Some red are flowers" (I-type conversion is valid) → TRUE ✓
Key Learning: All A are B + Some B are C = NO conclusion about A and C!
Answer: Only II follows ✓
Q.6. Statements: No fish is a bird. No bird is a horse.
Conclusions: I. No fish is a horse. II. Some horses are not fish. Ans: Neither conclusion follows
Method: Fish and Bird are separate. Bird and Horse are separate. But Fish and Horse COULD overlap or be separate - we don't know!
Rule: Two negative statements (E+E) give NO valid conclusion.
I. No fish is a horse: Possible but not certain (fish and horse could overlap) → FALSE ✗
II. Some horses are not fish: Not necessarily derivable → FALSE ✗
Answer: Neither follows ✓
Q.7. Statements: Some teachers are doctors. Some doctors are engineers.
Conclusions: I. Some teachers are engineers. II. Some engineers are doctors. Ans: Only Conclusion II follows
Method: Teacher overlaps Doctor. Doctor overlaps Engineer.
I. Some teachers are engineers: The overlap of Teacher-Doctor and Doctor-Engineer might be in DIFFERENT parts of Doctor. No guarantee → FALSE ✗
II. Some engineers are doctors: Direct conversion of "Some doctors are engineers" → TRUE ✓
Rule: I + I (two particular affirmatives) = No conclusion about the extreme terms. But conversion of given statement is always valid.
Answer: Only II follows ✓
Q.8. Statements: All boys are students. No student is a girl.
Conclusions: I. No boy is a girl. II. Some girls are not students. Ans: Only Conclusion I follows
Method: Boy is inside Student. Student and Girl are completely separate.
I. No boy is a girl: Boy is inside Student. Student has NO overlap with Girl. So Boy also has NO overlap with Girl → TRUE ✓
II. Some girls are not students: "No student is a girl" means they're completely separate. This means ALL girls are not students (stronger statement). Wait - Revised: "No student is a girl" = "No girl is a student" → This means ALL girls are not students → "Some girls are not students" is TRUE ✓
