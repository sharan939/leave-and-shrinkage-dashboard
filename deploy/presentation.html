  • No visibility into per-program productivity across associates

Before Dashboard

Managers spent ~40 min/week per report gathering data from SharePoint, calculating metrics, formatting emails, and tracking leaves across 20+ associates and 9 programs.

Feature Deep-Dive

WBR Narrative Email Generator

Transforms raw SharePoint data (458+ rows/month) into a structured, manager-style WBR email in seconds.

Input

  • Paste tab-separated data from SharePoint Excel
  • Or upload CSV/TSV file directly
  • Auto-detects columns: Login, HC, WK, Date, Program, Sub-Program, HC Utilization, Shrinkage Type, Target, Actual, Inflow, Backlog, Comments
  • Handles "Utilized"/"Shrinkage" text values
  • Week numbers auto-prefixed ("4" → "Week 4")
  • Filter by Week and/or Program before generating

Output (7-Section Email)

  • 1. Executive Summary (audits, target, achievement, shrinkage, backlog)
  • 2. Manager Brief (custom context/highlights)
  • 3. Weekly Trend (table with WoW comparison)
  • 4. Program-wise Performance (table with per-HC productivity)
  • 5. Per-Person Productivity (table)
  • 6. Shrinkage & Utilization (type breakdown)
  • 7. Actions & Next Steps (auto-generated)

Technical Details

Architecture & Integrations

Frontend

Pure HTML/CSS/JavaScript. No framework dependencies. Single-file deployment. Works offline via localStorage.

Vanilla JSCSS GridResponsive

Data Layer

localStorage for persistence. SharePoint Excel as source of truth. CSV/TSV import/export. Phonetool for org data.

localStorageSharePointCSV

Integrations

Phonetool (org sync), Outlook (HTML clipboard), SharePoint (data import), mailto: (reminders, WBR send).

PhonetoolOutlookmailto:

Key Design Decisions

Zero deployment: Open HTML file in browser. No server needed.

Offline-first: All data in localStorage. Works without network.

Multi-manager: Click through any manager in the org to see their data.
